Architecture

How it works

Flask serves the pages and streams annotated frames via MJPEG. All processing runs on your local machine.

01
🎥

Camera capture

OpenCV opens the default webcam and reads raw BGR frames at the device's native rate.

02
⚙️

Frame processing

Each frame is processed by the active mode — HSV masking, Haar cascade, or Tesseract OCR — with annotations drawn on top.

03
📡

MJPEG stream

Annotated JPEG frames are pushed to the browser over a multipart HTTP response. No WebSocket or WebRTC required.

04
💻

Browser UI

Single-page HTML renders the stream, sends control commands via a JSON API, and provides a snapshot button.

Get started

Running in minutes

No Docker, no cloud account, no API key. Clone the repo and go.

1

Clone the repository

Get the code from GitHub with git clone and cd into the folder.

2

Install Python dependencies

Five packages: flask, opencv-python, pillow, pytesseract, numpy.

3

Install Tesseract OCR engine

Only needed for text detection. Available via brew, apt, or the Windows installer linked in the README.

4

Start and open

Run python app.py and visit localhost:5000 in your browser.

TERMINAL
# 1. Clone git clone https://github.com/your-username/vision-ai cd vision-ai # 2. Python dependencies pip install flask opencv-python pillow pytesseract numpy # 3. Tesseract OCR engine # macOS: brew install tesseract # Ubuntu / Debian: sudo apt install tesseract-ocr # Windows: see github.com/UB-Mannheim/tesseract # 4. Run python app.py # Open: http://localhost:5000

Sharing

Share with others online

Two approaches to let people access your running instance from anywhere.

Free · Instant

ngrok tunnel

Expose your local server with a public HTTPS URL in one command. Perfect for demos and quick sharing.

TERMINAL
python app.py & ngrok http 5000 # Share the https://xxxx.ngrok.io URL
VPS / Cloud

Self-hosted server

Deploy on any Linux VPS (DigitalOcean, Linode, EC2). Use Gunicorn with a single worker since the camera is a shared resource.

TERMINAL
pip install gunicorn gunicorn -w 1 -b 0.0.0.0:5000 app:app # -w 1 required: camera is shared state
